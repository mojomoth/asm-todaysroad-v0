# 오늘의길 메인 지도 화면 컨텍스트

오늘의길은 사용자의 현재 위치를 중심으로 주변 산책 코스를 추천하고, 지도 위에 산책길과 추천 지점을 보여주는 모바일 앱이다.

이 화면은 앱의 메인 탐색 화면이다. 전체 화면은 지도이며, 사용자는 지도 위의 핀을 통해 주변 산책 코스를 발견한다. 특정 코스가 선택되면 하단 카드에서 코스명, 주소, 태그, 거리, 걸음 수, 예상 시간을 확인할 수 있다.

## 화면 구조

- 전체 화면 지도
- 상단 플로팅 헤더
- 지도 위 추천 핀
- 현재 위치 마커
- 하단 선택 코스 카드

## 상단 플로팅 헤더

상단에는 흰색 pill 형태의 플로팅 바가 있다.  
좌측에는 오늘의길 로고가 들어가고, 우측에는 카메라 아이콘과 브랜드/기능 아이콘이 배치된다.  
텍스트 없이 아이콘 중심으로 구성한다.  
지도 위에 떠 있는 느낌을 주되 그림자는 과하지 않게 한다.

## 지도

지도는 화면 전체를 채우며, 밝고 부드러운 톤이어야 한다.  
지도 정보는 보이지만 UI보다 튀지 않아야 한다.  
추천 장소는 코랄 레드 핀으로 표시하고, 현재 위치는 하늘색/파란색 마커로 표시한다.

## 마커

추천 지점 마커는 단순한 지도 핀 형태다.  
코랄 레드 컬러를 사용하고 중앙에 흰색 원을 둔다.

현재 위치 마커는 파란색 계열을 사용한다.  
중앙에 흰색 원이 있고, 아래에 작은 그림자 형태가 있어 현재 위치임을 강조한다.

## 하단 코스 카드

하단에는 선택된 산책 코스 카드가 떠 있다.  
카드는 흰색 배경, 큰 둥근 모서리, 부드러운 그림자를 가진다.

카드 구성:
- 좌측 원형 썸네일 이미지
- 우측 코스 제목
- 주소
- 태그 pill 리스트
- 하단 메트릭 영역

하단 메트릭:
- 거리
- 걸음 수
- 예상 시간

예시 데이터:
- 제목: 서울도보관광코스 한강순례
- 주소: 서울특별시 마포구 토정로 278-1
- 태그: #용강상점가, #YG사옥, #합정역
- 거리: 3.78km
- 걸음 수: 6,302걸음
- 예상 시간: 56분

## 디자인 방향

전체적으로 심플하고 밝은 지도 앱 UI를 유지한다.  
복잡한 기능을 한 화면에 노출하지 않는다.  
핵심은 “현재 위치 주변의 걷기 좋은 길을 발견하는 경험”이다.

둥근 모서리, 넉넉한 여백, 부드러운 그림자, 파스텔 계열 포인트 컬러를 사용한다.  
지도는 배경이고, 선택된 코스 카드가 핵심 정보 영역이다.

## 금지할 것

- 너무 많은 버튼을 추가하지 않는다.
- 지도 위에 텍스트를 과하게 올리지 않는다.
- 카드 안에 설명문을 길게 넣지 않는다.
- 마커를 복잡한 아이콘으로 만들지 않는다.
- 네비게이션 앱처럼 딱딱하고 기능적인 분위기로 만들지 않는다.
