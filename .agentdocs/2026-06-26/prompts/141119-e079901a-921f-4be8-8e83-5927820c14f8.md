# 프롬프트 (2026-06-26 14:11:19)

```
`.harness/v0/` 폴더에 하네스를 구성해줘. 하네스는 가이드와 가드레일이 있어야하고 루프 엔지니어링을 위한 랄프 루프도 있어야해

먼저 `/debate-loop` 명령어를 만들거야 입력이 되면 A2A에이전트 토론을 한다

에이전트는 아래 플러그인(스킬)에서 가져와서 생성한다 (.claude 참고)
- gstack(https://github.com/garrytan/gstack)
- mattpocock(https://github.com/mattpocock/skills)

debate loop는 moderator 에이전트와 devil 에이전트가 필요하다 moderator에이전트는 중재하고 회의를 진행한다. devil에이전트는 적대적 관찰자로 입력된 기획을 비판적으로 판단한다. 토론 루프의 마지막에는 devil 에이전트가 문제점을 제시하고 moderator 에이전트가 다시 문제점을 바탕으로 토론을 시작한다. 토론 루프는 3회 이상 수행하고 최종적으로 SPEC.md와 PRD.md 그리고 토론 DEBATE.md 를 출력한다.
```
